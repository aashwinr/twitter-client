# Twitter Bot

> ### About

- The twitter client works by pulling data from Twitter using the Twitter API.
- Currently it supports the following features:
  - Showing tweets, likes, and retweets.
  - Live update
  - Tweet Composition
  - Twitter Search
  - and more...
